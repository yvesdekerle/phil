# docs/design — matière pour la refonte design de Phil

Dossier de support pour la **refonte design** (exploration avec Fable 5 / Claude Design).

- **`donnees-exemple-maurice.md`** — jeu de données **fictif et cohérent** (voyage « Île Maurice 2026 », 9 personnes) pour **peupler les maquettes avec du contenu réaliste** : équipage, calendrier, carte (étapes Polarsteps), budget avec équilibres, idées, valise, repas, sondage, photos, coffre. **Version lisible.**
- **`donnees-exemple-maurice.json`** — la même chose, **structurée** (pour parser si besoin).

Contexte produit : voir `../FONCTIONNALITES.md` (catalogue des fonctionnalités) et `../../CLAUDE.md` (modèle de données, règles de partage, ton).

> Le **brief** et les **5 prompts** de design vivent hors dépôt (fichiers locaux, non commités). Les données ci-dessus, elles, sont commitées pour être lisibles par un outil qui parcourt le repo (local **ou** GitHub).
